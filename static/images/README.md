# Images Directory

Place your images here. They'll be accessible at `/images/filename.jpg` in your site.

## Recommended Images

- `profile.jpg` - Your profile photo (square, 400x400px minimum)
- `hero-bg.jpg` - Hero section background (1920x1080px)
- Project screenshots
- Blog post images
- Diagrams and illustrations

## Image Optimization Tips

- Use WebP format when possible for better compression
- Optimize images before uploading (use tools like TinyPNG)
- Keep file sizes under 500KB for fast loading
- Use descriptive filenames: `project-task-manager-screenshot.jpg`

## Example Usage in Markdown

```markdown
![Alt text](/images/your-image.jpg)
```
