================================================================================
USAGE INSTRUCTIONS
================================================================================

FOR STUDENTS:
1. Fork this repository
2. Enable GitHub Pages (Settings → Pages → GitHub Actions)
3. Edit hugo.toml with your information
4. Push changes - site deploys automatically
5. Visit your live site at yourusername.github.io

FOR INSTRUCTORS:
1. Review INSTRUCTOR_GUIDE.md for lesson plans
2. Use 90-minute workshop outline
3. Follow teaching tips and rubrics
4. Help students troubleshoot common issues
5. Assess using provided rubrics

FOR DEVELOPERS:
1. Clone repository
2. Run: git submodule update --init --recursive
3. Install Hugo
4. Run: hugo server -D
5. Customize and contribute
